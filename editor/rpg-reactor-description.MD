# RPG Reactor

The free, open-source RPG engine built for creators who want more.

RPG Reactor is a complete game development studio for building RPGs, the kind of tool you'd expect to pay for, except it's free, it's open source, and it's yours to keep forever.

If you've ever hit a wall with RPG Maker's limitations, or wished you could crack open the editor itself and reshape it, RPG Reactor was made for you.

It's free. Really free. MIT licensed. No royalties, no subscriptions, no features locked behind DLC. The full editor, the full engine, the full source code. Use it for commercial games, personal projects, game jams — whatever you want.

It speaks your language. If you know RPG Maker, you already know how to use RPG Reactor. Same concepts, same workflow, same terminology but without the constraints. Already have an RPG Maker MZ project? Import it directly and pick up right where you left off. Maps, events, database, plugins, assets — everything carries over.

It runs everywhere you do. Build on Windows, macOS, Linux or the Web.

Everything you need to make your game:

- Full-featured map editor
- Visual event system — no coding required
- Built-in combat system
- Complete database editors
- Plugin support
- Audio tools
- Multi-window project editing with same-project lock protection
- Cross-instance copy/paste for events, event commands, and maps
- One-click playtesting
- One-click cross-platform builds

It's all here. And when you want to go beyond what the visual tools offer, the full JavaScript engine is wide open — no obfuscation, no locked internals, no guessing how things work.

For RPG Maker developers:

- Import your existing MZ projects. Open your project folder directly in RPG Reactor. Everything carries over. No conversion tools, no export steps, no data loss. Pick up right where you left off.
- No cost barrier. Full engine, no tiers, no features behind a paywall.
- Cross-platform out of the box. Native Linux support as a first-class citizen. Build for Windows, macOS, Linux, and Web from any platform.
- See how everything works. The entire editor and engine source code is open. If something doesn't work the way you want, change it.
- Familiar workflow. Your existing knowledge transfers directly.
- Keep your plugins. MZ plugins work in RPG Reactor. Your plugin library isn't going anywhere.

For new developers:

- Visual everything. Map painting, event scripting, database editing — all through intuitive interfaces. No programming knowledge needed.
- Fast editor workflows. Preview map BGM/BGS in place, pick actor face/sprite cells visually, and use scoped database shortcuts without map or event tools interfering.
- Start right away. Ships with a starter template so you can jump in and start creating from the moment you open it.
- Built for long sessions. Dark-themed editor designed to be easy on your eyes.
- Go as deep as you want. Start with the visual tools, and when you're ready, the full engine is right there waiting.



Open source and community-driven.

RPG Reactor is built in the open, and the roadmap is shaped by the people who use it. Report bugs, suggest features, submit code, build plugins, share assets, write tutorials — this project grows with its community.

This isn't a closed product with a suggestion box. The source code is on GitHub, the license is MIT, and the door is always open. Whether you want to fix a bug, add a feature, or build something entirely new on top of the engine — you can.

RPG Reactor is in active development and already capable enough to build and ship real games. New features and improvements land regularly, and the roadmap is driven by what creators actually need. This is a living project, and it's moving fast.


RPG Reactor is made by RPG developers, for RPG developers. Whether you're a veteran shipping your tenth game or a newcomer placing your first tile, there's a place for you here.

Download it, start creating, and if you like what you see — join the community and help shape where this goes next.

The best RPG engine is one that belongs to its creators. Let's build it together.
